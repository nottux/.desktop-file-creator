this script creates a .desktop file like:
[Desktop Entry]
Name=
Exec=
Terminal=true
Type=Application
Comment=
Icon=
it asks you to add comment and icon line to be added and running in terminal to be true or false
you can also create $HOME/desktop_file_locations.txt to add short names of locations like
utku="/home/utku/Masaüstü/Uygulamalar"
somefolder="/home/utku/some folder"
